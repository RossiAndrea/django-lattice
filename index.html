Overview

Django-lattice is a reusable Django application for generating a basic scaffold for a CMS.
It includes a full set of abstract models, useful for an effort-less creation of site content models, like ThumbnailAbstract that implements a simple thumbnail for the model, with a premade function bindend to the post-save event which crop and resize the uploaded image to the given size.

This application includes also several useful management commands that speeds up the development process, and gives to you a basic scaffold to create tasty layouts. For example, if you'd like to use Foundation you only have to type python manage.py getfoundation to download and sync the entire front-end framework into your Django project!

Dependencies

Django-lattice makes use of as few library as possible. The requirements are the following:

  • Python 2.6+
  • Django 1.5+ (untested on older versions)
  • Pillow
  • Python Requests

Browser support

Django-lattice has no particular client-side requirements. By default every browser (even the older ones) should work just fine.

Beware: if you intend to use "Foundation" (or maybe Bootstrap in a near future...) consider the framework support (usually these frameworks work with most of modern browser, just older versions of IE should be problematic)

Installation

The easiest method is to install directly from pypi using pip by running the command below, which will also install all the required dependencies:

pip install git+https://github.com/RossiAndrea/django-lattice.git

If you prefer, you can download Django-lattice and install it directly from source:

  • git clone https://github.com/RossiAndrea/django-lattice.git
  • python setup.py install

Or simply copy the lattice directory to your django project root.
Once done add lattice to the INSTALLED_APP set in settings.py. For example:

  1. INSTALLED_APPS = (
  2.     'django.contrib.admin',
  3.     'django.contrib.auth',
  4.     'django.contrib.contenttypes',
  5.     'django.contrib.sessions',
  6.     'django.contrib.messages',
  7.     'django.contrib.staticfiles',
  8.     'django.contrib.sites',
  9.     'lattice'
  10. )

Beware: using the last two methods you'll need to install manually all the dependencies. Using pip you can install everything just typing: pip install -r requirements.txt. The requirements.txt is located in the repository root.
